Hackathon Starter

A boilerplate for Golang web applications. Optimized for developer speed.

Based on Gobuffalo

Gobuffalo is a Go web development eco-system, designed to make the life of a Go web developer easier. Buffalo uses Gorilla toolkit to manage routes, sessions, cookies... There might be faster routers out there, but this one is definitely the most powerful!

Supports Account Management

Supports Github, Twitter, Facebook login out of the box. Additional logins can be added with a one-liner using goth. Also has account management built in.

Hot Code Reload

Code, save, refresh. Use the buffalo dev command to rebuild your app, from backend to frontend, and just see the changes live!

API Examples

Comes with API examples for most common tasks like scraping, file upload, integrating APIs for twitter etc
